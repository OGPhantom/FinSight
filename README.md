# 💸 FinSight
A **privacy-first, on-device AI-powered personal finance tracker for iOS** built with **SwiftUI**, **SwiftData**, **Charts**, and **Apple Intelligence (FoundationModels)**.
Track your **daily spending** and generate **AI-powered financial reports** with insights, trends, and actionable recommendations.

---

### 📌 Short Description

FinSight is a premium iOS expense tracking app focused on **AI-powered financial reports**, **spending analytics**, and **deep customization**.  
Users can log daily expenses, manage fully customizable categories, explore visual analytics, and generate AI-based reports for selected periods, all processed **entirely on-device for maximum privacy**.
The app also allows broad personalization of the experience, from **custom category colors and icons** to **themes, accent colors, dynamic background appearance and currencies**, creating a finance tool that feels both personal and intelligent.

---

### 🚀 Why FinSight

– 🧠 Fully on-device AI — your financial data never leaves your device  
- ⚡ Fast, privacy-first insights powered by Apple Foundation Models
- 🎯 Flexible reporting — from daily tracking to long-term analysis

## ✨ Features

- 🤖 **AI-powered financial reports** with **spending insights, trends and actionable recommendations**
- 📆 Generate reports for **preset periods** or a **custom date range**
- 💸 Add, edit, delete, search and sort **expense transactions**
- 🗂 Create, customize and delete **custom categories**
- 🔄 Safely delete categories **without losing** existing transactions
- 📊 Explore analytics with **spending trends, breakdowns, top merchants, and period-based insights**
- 💱 Choose your **preferred currency** for app-wide financial formatting
- 🎨 Personalize the app with **theme and accent color selection**
- 💾 Store **transactions, categories, and reports** locally with SwiftData
- 🌗 Enjoy full **Light and Dark Mode** support
- 🗑 Reset **transactions, reports, and custom categories** in one tap

## 📄 Requirements

- Xcode 16+
- iOS 18+
- Apple Intelligence-supported device or simulator setup for AI reporting features

---

## 🚀 Installation

1. Clone the repository.
2. Open `FinSight.xcodeproj` in Xcode.
3. Build and run on an iPhone simulator or device.
4. For AI-generated reports, use an environment that supports Apple Intelligence / FoundationModels.

---

## 🧠 Notes

- Transactions and categories are stored locally with SwiftData.
- AI report generation depends on Apple Intelligence availability on the current device.
- If the AI model is unavailable, the rest of the app remains fully functional for tracking and analytics.
